# Frontend Day 2: Bootstrap Training Labs 🚀

Repository นี้รวบรวมไฟล์แบบฝึกหัด (Labs) สำหรับการเรียนรู้และพัฒนา Frontend ด้วย **Bootstrap** (Day 2) เนื้อหาในแบบฝึกหัดมุ่งเน้นไปที่การสร้าง User Interface ที่ใช้งานได้จริง โดยสอดแทรกแนวคิด **Industrial UI/UX Design** ที่ให้ความสำคัญกับการตัดกันของสีอย่างชัดเจน (High Contrast) และความสามารถในการกวาดตามองเพื่อรับรู้ข้อมูลได้อย่างรวดเร็ว (Glanceability) ซึ่งเป็นหัวใจสำคัญสำหรับการออกแบบหน้าจอแดชบอร์ดหรือหน้าจอควบคุมในสภาพแวดล้อมของโรงงานอุตสาหกรรม

## 📂 โครงสร้างของโปรเจกต์ (Project Structure)

โปรเจกต์นี้ประกอบไปด้วยไฟล์แบบฝึกหัด HTML จำนวน 7 ไฟล์ ซึ่งถูกออกแบบมาให้ผู้เรียนได้ฝึกฝนการใช้ Grid System และ Components ต่างๆ ของ Bootstrap ตามลำดับความซับซ้อน:

* `lab1.html` - แบบฝึกหัดพื้นฐาน โครงสร้างเริ่มต้น
* `lab2.html` - แบบฝึกหัดที่ 2
* `lab3.html` - แบบฝึกหัดที่ 3
* `lab4.html` - แบบฝึกหัดที่ 4
* `lab5.html` - แบบฝึกหัดที่ 5
* `lab6.html` - แบบฝึกหัดที่ 6
* `lab7.html` - แบบฝึกหัดประยุกต์ขั้นสูง

## 🛠️ เทคโนโลยีที่ใช้ (Technologies Used)

* **HTML5** สำหรับวางโครงสร้าง Semantic ของหน้าเว็บ
* **Bootstrap** (CSS Framework) สำหรับจัดการ Layout และ Components อย่างรวดเร็ว

## 🚀 วิธีการใช้งานและการรันโปรเจกต์ (How to Run)

Clone repository นี้ไปยังเครื่องคอมพิวเตอร์ของคุณ:
   ```bash
   git clone [https://github.com/gatunyus/Frontend_Day2_Bootstrap.git](https://github.com/gatunyus/Frontend_Day2_Bootstrap.git)
   ```
เปิดโฟลเดอร์โปรเจกต์ขึ้นมา
คุณสามารถรันไฟล์ `lab1.html` ถึง `lab7.html` ได้ง่ายๆ โดยการดับเบิลคลิกเพื่อเปิดผ่าน Web Browser โดยตรง
**คำแนะนำเพิ่มเติม:** หากคุณเขียนโค้ดด้วยโปรแกรม Text Editor อย่าง **Antigravity** (หรือ VS Code) แนะนำให้ติดตั้งและเปิดใช้งาน Extension **Live Server** เพื่อให้หน้าเว็บทำการ Refresh อัตโนมัติทุกครั้งที่มีการบันทึกการแก้ไขโค้ด
