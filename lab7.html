กล่อง (Box Model) 📦

ความแตกต่างระหว่างไม่มีอะไรเลย กับการใช้ Padding (ขอบใน), Border (เส้นขอบ), และ Margin (ขอบนอก)

❌ ไม่ใช้ CSS (ดิบๆ)
คำเตือน: ห้ามเข้าใกล้เครื่องจักรขณะทำงาน
กล่องใบที่สอง (ติดกันเลย)
✅ ใช้ CSS (Box Model)
คำเตือน: ห้ามเข้าใกล้เครื่องจักรขณะทำงาน
กล่องใบที่สอง (ถูก Margin ผลักออกไป)

การจัดรูปแบบตัวอักษร (Typography) 📝

การเปลี่ยนฟอนต์ ขนาด สี และการจัดกึ่งกลาง

❌ ไม่ใช้ CSS
บริษัท เสริมสุข จำกัด (มหาชน)
✅ ใช้ CSS (Typography)
บริษัท เสริมสุข จำกัด (มหาชน)

เงาและความโค้งมน (Modern Design) ☁️

เปลี่ยนกล่องสี่เหลี่ยมแข็งๆ ให้ดูเป็นหน้าต่างแอปพลิเคชันยุคใหม่

❌ ไม่ใช้ CSS
ตารางสรุปผลการผลิต (กล่องแบนๆ ขอบเหลี่ยม)
✅ ใช้ CSS (Radius & Shadow)
ตารางสรุปผลการผลิต (ขอบมนและมีเงาดูลอยขึ้นมา)

ลูกเล่นโต้ตอบกับผู้ใช้งาน (Hover Effects) 🖱️

ลองเอาเมาส์ไปชี้ที่ปุ่มทั้ง 2 ฝั่งเพื่อดูความแตกต่าง

❌ ไม่ใช้ CSS (แข็งทื่อ)
✅ ใช้ CSS (:hover + transition)
